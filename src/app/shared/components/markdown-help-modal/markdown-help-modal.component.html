Markdown syntax

VtesDecks descriptions and comments are written using Markdown, a plain-text formatting language. However, we also add in some of our own flavor to allow you to do special things (e.g., adding card, discipline, clans, etc). We also removed some of the custom HTML from Markdown to prevent people from abusing the design of the site.

This article will show you some of the basics of creating a markdown text.

Editor Basics

Previewing — While you're editing, you can press the Preview button in the toolbar to see what your text will look like after processes all of the Markdown.

Formatting Text

Headers — Headers are useful for organizing your content and making the text more organized and readable. To create a header, use the number sign (#) followed by your header title. The number of number signs you use will dictate the size of the header that is rendered. One being the largest and six being the smallest.

For example, ## This is a header would produce the following:

This is a header

#### This is a smaller header would produce the following:

This is a smaller header

Be sure to include an empty line above and below your header.

Bold — In order to bold your text, surround the text with two asterisks (i.e., to make Bold Text you would type **Bold Text**).

Italic — In order to italicize your text, surround the text with one underscore (i.e., to make Italic Text you would type _Italic Text_).

Strikethrough — In order to strikethrough your text, surround the text with two tildes (i.e., to make Strikethrough Text you would type ~~Strikethrough Text~~).

Lists

Unordered Lists — Unordered lists are lists that use bullets to start an indented line. To create an unordered list, type an dash (-) at the beginning of a line followed by the list item. An example would look like this:

  - Unordered List Item 1
  - Unordered List Item 2
  - Unordered List Item 3

If you wish to create a nested unordered list, you may do so by adding 4 spaces (avoid using tab indentation) preceding the list item on subsequent lines. An example would look like this:

  - Unordered List Item 1
  - Unordered List Item 2
      - Indented List Item 1
      - Indented List Item 2
  - Unordered List Item 3

Ordered Lists — Ordered lists are lists that use numbers followed by a period to start an indented line. The list will begin at whatever number you choose followed by a period. Each subsequent line that begins with any number less than 10 digits long followed by a period will create another sequential entry in your ordered list regardless of what number you've chosen. To create an ordered list, type a number at the beginning of a line followed by a period and a space then the list item. An example would look like this:

  1. Ordered List Item 1
  2. Ordered List Item 2
  3. Ordered List Item 3

If you wish to create a nested ordered list, you may do so by adding 4 spaces (avoid using tab indentation) preceding the list item on subsequent lines. An example would look like this:

  1. Ordered List Item 1
  2. Ordered List Item 2
      1. Indented List Item 1
      2. Indented List Item 2
  3. Ordered List Item 3

You may also nest ordered lists into unordered lists and vice versa. An example would look like this:

  1. Ordered List Item 1
  2. Ordered List Item 2
      - Indented Unordered List Item 1
          1. Indented Ordered List Item 1
          2. Indented Ordered List Item 2
              - Indented Unordered List Item 1
  3. Ordered List Item 3

Embedding Links, Images and Videos

Links — To create a link, surround some text with brackets followed by the URL surrounded with parentheses. For example, to create a link to VtesDecks you would type [VtesDecks](https://vtesdecks.com).

You may also add an optional title to a link by enclosing some text in quotation marks after the URL. This will result in in a tooltip when the user hovers over the link. Adding to our previous example, the result of [VtesDecks](https://vtesdecks.com "A deck building website for Vampire: The Eternal Struggle") would look like this: VtesDecks

Images — To create an image, begin with an exclamation mark (!) followed by some alt text in brackets then the URL to the image asset in parentheses. For example, ![My Favorite GIF](https://media1.giphy.com/media/giphy.gif)) will result in the following:

If you wish to make an image into a link, enclose the markdown for the image in brackets, and then add the URL in parentheses. For example, [![VtesDecks](https://vtesdecks.com/assets/img/logo.png)](https://vtesdecks.com) will result in the following:

YouTube Videos — To create an embedded YouTube video, surround with double brackets the word youtube (all lower case) followed by a colon and then the link ID to the YouTube video you want to embed. The link ID can be found after https://youtu.be/ in the URL when selecting share from the YouTube video or copied from your browser's address bar. For example, the URL https://youtu.be/IxX_QHay02M includes the link ID IxX_QHay02M. For example, [[youtube:IxX_QHay02M]] will result in the following:

Embedding Cards

Cards — Embedding a card will create a link that, when hovered over by the user, will display the card. To embed a card, surround the card's name in double brackets. The format is not case sensitive, however, any punctuation in the card's name must be present. For example, [[card:British Museum, London, The]] with result in the following:

Embedding Disciplines

Discipline — Many of Vampire: The Eternal Struggle's disciplines symbols are available for use in the editor. The format for manually adding the disciplines is surrounding with double brackets the word discipline (all lower case) followed by a colon and then the discipline abbreviation or name. For example, [[discipline:Animalism]] with result in the following:

The easiest way to add discipline symbols to your text is by using the button on the toolbar and then selecting the discipline of your choice.
